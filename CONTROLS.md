# Contrôles du Retro Gaming Console

## 🎮 Contrôles Clavier

### Joueur 1 (Clavier principal)
- **Directionnel** : Flèches (↑ ↓ ← →)
- **Start** : `S`
- **Select** : `A`
- **Bouton A** : `X`
- **Bouton B** : `Z`

### Joueur 2 (Pavé numérique)
- **Directionnel** : 8, 2, 4, 6 (Numpad)
- **Start** : 3 (Numpad)
- **Select** : 1 (Numpad)
- **Bouton A** : 7 (Numpad)
- **Bouton B** : 9 (Numpad)

## � Menu de Jeu

### Ouvrir le menu
- **Bouton menu** : Cliquer sur le bouton en haut à gauche (☰)
- **Raccourci clavier** : `Select (A)` + `Start (S)` simultanément
- **Fermer** : Cliquer en dehors du menu ou appuyer sur `ESC`

### Fonctionnalités du menu
- **🔊 Volume** : Contrôle du volume audio (0-100%)
- **💾 Sauvegarder l'état** : Crée un point de sauvegarde
- **📂 Charger l'état** : Restaure le dernier point de sauvegarde
- **🔄 Redémarrer** : Redémarre le jeu depuis le début
- **⛶ Plein écran** : Active/désactive le mode plein écran
- **← Retour/Quitter** : Retourne à la liste des jeux

**Note** : Le jeu se met automatiquement en pause quand le menu est ouvert.

## 🔧 Formats Supportés

Le système supporte les fichiers compressés `.zip` qui sont automatiquement extraits en mémoire.

## 📱 Contrôles Tactiles

Sur mobile, des contrôles tactiles à l'écran sont automatiquement affichés.

## Émulateurs Supportés

| Émulateur | Extensions acceptées | Statut |
|-----------|---------------------|--------|
| Nintendo (NES) | .nes, .zip | ✅ Activé |
| Super Nintendo (SNES) | .snes, .sfc, .zip | ✅ Activé |
| Nintendo 64 | .n64, .zip | ✅ Activé |
| Game Boy / GBC | .gb, .gbc, .zip | ✅ Activé |
| Game Boy Advance | .gba, .zip | ✅ Activé |
| Sega Genesis | .gen, .smd, .bin, .zip | ✅ Activé |
| MS-DOS | .dos, .iso, .img, .bin, .zip | ❌ Désactivé |
| MAME Arcade | .zip | ❌ Désactivé |

## Navigation
- **Page d'accueil** : Liste tous les émulateurs avec le nombre de jeux
- **Page émulateur** : Affiche tous les jeux disponibles pour cet émulateur
- **Player** : Interface plein écran avec menu intégré
