# Contrôles du Retro Gaming Console

## 🎮 Contrôles Clavier

### Joueur 1 (Clavier principal)
- **Directionnel** : Flèches (↑ ↓ ← →)
- **Start** : `S`
- **Select** : `A`
- **Bouton A** : `X`
- **Bouton B** : `Z`

### Joueur 2 (Pavé numérique)
- **Directionnel** : 8, 2, 4, 6 (Numpad)
- **Start** : 3 (Numpad)
- **Select** : 1 (Numpad)
- **Bouton A** : 7 (Numpad)
- **Bouton B** : 9 (Numpad)

## 🔊 Contrôle du Volume
Un slider de volume est disponible en haut à droite de l'écran de jeu pour ajuster le son de 0% à 100%.

## Émulateurs Supportés

| Émulateur | Extensions acceptées |
|-----------|---------------------|
| DOS | .dos, .iso, .img, .bin, .zip |
| GameBoy | .gb, .gbc, .zip |
| GameBoy Advance | .gba, .zip |
| Genesis | .gen, .smd, .bin, .zip |
| MAME | .zip |
| Nintendo (NES) | .nes, .zip |
| Nintendo 64 | .n64, .zip |
| Super Nintendo | .snes, .zip |

## Navigation
- **Page d'accueil** : Liste tous les émulateurs avec le nombre de jeux
- **Page émulateur** : Affiche tous les jeux disponibles pour cet émulateur
- **Player** : Interface plein écran avec contrôles et volume
  - Bouton "← Retour" en haut à gauche pour revenir à la liste
  - Volume en haut à droite
